## Project 6 - PIG (Game) (Medium)

The Pig Game is a simple dice game usually played by two or more players. The goal is to be the first player to reach a set score. On each turn, a player rolls a die as many times as they want, adding the results to their turn score. However, if the player rolls a 1, their turn ends and they lose all points earned during that turn.

## Project 7 - Mad Libs generator (Medium)
